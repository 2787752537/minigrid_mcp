# MiniGrid Play Only

This project only keeps a minimal launcher for the official graphical MiniGrid game.

## Run

Use the Conda environment you already created:

```powershell
D:\Anaconda\Anaconda\envs\minigrid-agent\python.exe D:\code\codex_2\minigrid_agent\play_minigrid.py
```

Examples:

```powershell
D:\Anaconda\Anaconda\envs\minigrid-agent\python.exe D:\code\codex_2\minigrid_agent\play_minigrid.py --env-id MiniGrid-Empty-8x8-v0
D:\Anaconda\Anaconda\envs\minigrid-agent\python.exe D:\code\codex_2\minigrid_agent\play_minigrid.py --env-id MiniGrid-DoorKey-8x8-v0
D:\Anaconda\Anaconda\envs\minigrid-agent\python.exe D:\code\codex_2\minigrid_agent\play_minigrid.py --env-id MiniGrid-Dynamic-Obstacles-8x8-v0
```
